MyCountdown allows you to create countdowns based on a start and end date. When you create a countdown, you'll get a unique link that you can share. Dates are based on your current timezone

This app was inspired by the Donut of Misery that many soldiers create to count the remainder of their deployments. It is a securtity violation to share any deployment or return dates, so this project is just for fun.

This project was created by Reagan Henke, and more information is available on Github
